1. Set pancake mix and water as variables
2. Mix pancake mix with water
3. Set griddle as a variable
4. Heat griddle
5. Set batter as a variable
6. Pour a circle of batter on the griddle
7. Set bubbles as a variable
8. Wait until most of the bubbles pop on the surface
9. Flip the pancake
10. Cook to desired color
11. Set butter and syrup as variables
12. Apply butter and syrup to the pancake
13. Enjoy the pancake

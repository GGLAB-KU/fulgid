1. Set rock as magma from a volcano
2. Let rock cool
3. Crystals form
4. Set rock as igneous rock
5. Let igneous rock break down into sediment
6. Set sediment as sedimentary rock
7. If subjected to heat and/or pressure from plate-tectonic activity, set sedimentary rock as metamorphic rock
8. If metamorphic rock melts, set rock as magma again.

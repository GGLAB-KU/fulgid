1. Begin erosion process of rocks by wind and water.
2. Pick up sediment that came off the rocks.
3. Carry sediment to an area using wind, water, air, or ice.
4. Deposit sediment on the same spot multiple times.
5. Allow sediment to pile on top of other sediment for a long time.
6. Compress sediment near the bottom with the weight of newer sediment.
7. Push sediment together with heavy weight to form sedimentary rock.

1. Set temperature variable to warm
2. If temperature is warm, water evaporates
3. Water rises up in the air
4. If there is a tropical storm forming, it increases the amount of water evaporated into the clouds
5. Set temperature variable to cool
6. If temperature is cool, water condenses
7. Condensed water forms into clouds
8. Check moisture level in clouds
9. If there is enough moisture, it starts to rain.

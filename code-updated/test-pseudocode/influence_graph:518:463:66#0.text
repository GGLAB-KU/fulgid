1. AdultFish lays Eggs
2. If more eggs are laid, increase the number of eggs
3. Eggs incubate for a few weeks or months
4. Eggs hatch into YoungFish
5. If more eggs were laid, increase the number of YoungFish
6. YoungFish remain in hatching area until they are big enough to venture into open water
7. Fish continue to grow
8. Fish reach adulthood
9. AdultFish reproduce, laying eggs and continuing the cycle.
10. If more eggs were laid, repeat the cycle with an increased number of eggs and YoungFish.

1. Initialize egg variable as "unhatched"
2. Wait for egg to hatch
3. If egg hatches, set tadpole variable as "born"
4. Tadpole swims with siblings and avoids predators
5. Tadpole grows larger over time
6. When tadpole reaches a certain size, it may lose its tail
7. If more tadpoles lose their tails, it may affect the development of limbs in less tadpoles
8. Tadpole continues to develop and may develop limbs
9. When tadpole has fully developed into a frog, set frog variable as "formed"

1. Bee visits flower
2. Pollen sticks to bee
3. Bee flies to another flower of the same type
4. Pollen on bee brushes onto pistil of flower
5. Flower is pollinated
6. If there are less bees to cross-pollinate, repeat steps 1-5 until all flowers are visited
7. If some flowers are not visited, they will not be pollinated and will not develop seeds and fruit.

1. For each bee:
2. Bee lands on flower
3. Bee's leg hairs attach to pollen sacs from stamens of flower
4. Bee flies away with pollen
5. Bee lands on new flower
6. Pollen sacs are rubbed off onto stigma of new flower
7. Repeat steps 2-6 for each bee
8. More bees increase the chances of pollination for more flowers.

1. Initialize tadpole
2. While tadpole has no hind legs:
   a. Feed tadpole
   b. Monitor growth
   c. If hind legs appear, go to step 3
3. While tadpole has a tail:
   a. Feed tadpole
   b. Monitor growth
   c. If tail is lost, go to step 4
4. Initialize frog
5. While frog is alive:
   a. Mate with another frog
   b. Produce offspring
   c. If more pollen is distributed to other areas, go to step 6
   d. If no more offspring are desired, end program
6. Increase the number of tadpoles hatched due to increased pollen distribution
7. Go to step 1 to hatch a new tadpole and begin the cycle again.
